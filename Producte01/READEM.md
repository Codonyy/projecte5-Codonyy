# P01: Control de versions. GitHub Classroom i git

## Breu descripció

Al llarg dels diferents projectes que hem anat treballant, GitHub ha esdevingut el lloc on presenteu la documentació. Malgrat que ja heu lliurat diversos projectes, és cert que documentar directament des de la web té una sèrie d’inconvenients: és lent, sobretot a l’hora de crear carpetes, pujar imatges, etc.

Per aquest motiu, en aquest projecte donarem una passa més en el món del control de versions. A partir d’ara treballarem bàsicament en local, farem versionat aprofitant la potència del protocol Git i sincronitzarem el repositori local amb el repositori de GitHub.

A més, solucionarem un problema que segurament ja us heu plantejat: fins ara, els vostres repositoris eren públics, de manera que és impossible controlar si algú decideix aprofitar-se de la vostra feina. Per això van néixer els repositoris privats, que només són visibles pels usuaris autoritzats. D’aquesta manera assegurem la confidencialitat de la nostra feina.

Com hi treballarem? Amb una plataforma de GitHub anomenada **GitHub Classroom**, que permet a l’àmbit educatiu disposar d’una infraestructura similar a la de les organitzacions: els repositoris són propietat de l’organització. Aquesta permet que cada treballador tingui un *fork* (amb Classroom es fa automàticament) del repositori base. Així, cadascun de vosaltres treballa amb un repositori propi i privat.

Més endavant farem que aquests repositoris passin a ser públics, de manera que podreu fer un fork al vostre compte de GitHub personal i afegir aquest projecte al vostre portfoli.
