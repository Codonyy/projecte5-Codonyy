# T06: Configuració del domini

## Breu descripció
Aquesta tasca se centra en la configuració d’un domini Active Directory un cop ja ha estat creat. L’objectiu és desplegar correctament els objectes del domini i comprovar el seu funcionament en un entorn client real.

## Introducció
Un cop disposem del domini creat, el següent pas és el seu desplegament lògic. Aquest procés inclou la creació i organització dels diferents objectes que formen el domini, com ara unitats organitzatives (OU), grups, usuaris i equips.

L’ús correcte de les unitats organitzatives permet una millor administració del domini, facilita l’aplicació de polítiques i millora l’escalabilitat i manteniment del sistema.
